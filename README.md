# Ruishi Chen - Personal Website

This is the source code for my personal academic website, built with Jekyll and hosted on GitHub Pages.

## About

I am a Ph.D. student in the Education Data Science program at Stanford University. This website showcases my research, publications, and academic activities.

## Website

Visit my website at: [https://ruishichen.github.io](https://ruishichen.github.io)

## Local Development

To run this site locally:

1. Install Ruby and Bundler
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Visit `http://localhost:4000`
