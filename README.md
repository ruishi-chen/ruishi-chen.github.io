## Local Development

### Option 1: Using Ruby/Bundler (Traditional)

To run this site locally with Ruby:

1. Install Ruby and Bundler
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Visit `http://localhost:4000`

### Option 2: Using Docker (Recommended)

To run this site locally with Docker:

1. Clone this repository
2. Make sure Docker and Docker Compose are installed
3. Run `docker-compose up`
4. Visit `http://localhost:4000`

The Docker setup includes live reload, so changes will automatically refresh in your browser.
