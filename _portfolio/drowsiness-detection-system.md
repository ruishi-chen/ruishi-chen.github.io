## Project Overview
Architected a sophisticated multi-modal attention monitoring system by deploying LeNet-5 CNN with fine-tuned MobileNetV2, achieving 95.31% accuracy on 4,200 synthetic eye images for real-time student attention monitoring.

## Key Achievements
* **High Accuracy**: Achieved 95.31% accuracy in drowsiness detection
* **Large Dataset**: Processed 4,200 synthetic eye images
* **Advanced Architecture**: Combined LeNet-5 CNN with fine-tuned MobileNetV2
* **Real-time Processing**: Implemented system for live attention monitoring
* **Computer Vision**: Leveraged Depth-wise Separable Convolutions for efficiency
